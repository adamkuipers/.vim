Contributing ....................... |Powerline-contributing|</div><div class='line' id='LC36'><br/></div><div class='line' id='LC37'>==============================================================================</div><div class='line' id='LC38'>1. Introduction                           *Powerline* *Powerline-introduction*</div><div class='line' id='LC39'><br/></div><div class='line' id='LC40'>Powerline is a utility plugin which allows you to create better-looking, more </div><div class='line' id='LC41'>functional vim statuslines.</div><div class='line' id='LC42'><br/></div><div class='line' id='LC43'>==============================================================================</div><div class='line' id='LC44'>2. Usage                                                     *Powerline-usage*</div><div class='line' id='LC45'><br/></div><div class='line' id='LC46'>Powerline is automatically enabled when it&#39;s installed, either by unzipping </div><div class='line' id='LC47'>the provided archive or by adding it as a Pathogen/Vundle bundle.</div><div class='line' id='LC48'><br/></div><div class='line' id='LC49'>==============================================================================</div><div class='line' id='LC50'>3. Requirements                                       *Powerline-requirements*</div><div class='line' id='LC51'><br/></div><div class='line' id='LC52'>Powerline has been developed and tested in vim 7.3, but it should run without </div><div class='line' id='LC53'>any problems in vim 7.2. The default configuration requires a Unix-like system </div><div class='line' id='LC54'>to work properly.</div><div class='line' id='LC55'><br/></div><div class='line' id='LC56'>The plugin only works with vim running in an 88/256-color terminal or gvim.</div><div class='line' id='LC57'><br/></div><div class='line' id='LC58'>Vi-compatible mode must be disabled.</div><div class='line' id='LC59'><br/></div><div class='line' id='LC60'>------------------------------------------------------------------------------</div><div class='line' id='LC61'>3.1 Recommended settings                      *Powerline-recommended-settings*</div><div class='line' id='LC62'><br/></div><div class='line' id='LC63'>The following configuration options should be set in your |vimrc|: &gt;</div><div class='line' id='LC64'><br/></div><div class='line' id='LC65'>&nbsp;&nbsp;&nbsp;&nbsp;set nocompatible   &quot; Disable vi-compatibility</div><div class='line' id='LC66'>&nbsp;&nbsp;&nbsp;&nbsp;set laststatus=2   &quot; Always show the statusline</div><div class='line' id='LC67'>&nbsp;&nbsp;&nbsp;&nbsp;set encoding=utf-8 &quot; Necessary to show unicode glyphs</div><div class='line' id='LC68'><br/></div><div class='line' id='LC69'>Note: If you&#39;re using an 88/256-color terminal but still don&#39;t see the colored </div><div class='line' id='LC70'>statusline, you may have to set the following option as well: &gt;</div><div class='line' id='LC71'><br/></div><div class='line' id='LC72'>&nbsp;&nbsp;&nbsp;&nbsp;set t_Co=256 &quot; Explicitly tell vim that the terminal supports 256 colors</div><div class='line' id='LC73'><br/></div><div class='line' id='LC74'>==============================================================================</div><div class='line' id='LC75'>4. Configuration                                     *Powerline-configuration*</div><div class='line' id='LC76'><br/></div><div class='line' id='LC77'>Powerline will work fine without any user configuration, but default behavior </div><div class='line' id='LC78'>can be overridden by setting configuration variables globally in your |vimrc| </div><div class='line' id='LC79'>file.</div><div class='line' id='LC80'><br/></div><div class='line' id='LC81'>------------------------------------------------------------------------------</div><div class='line' id='LC82'>4.1 Powerline_cache_file                                *Powerline_cache_file*</div><div class='line' id='LC83'><br/></div><div class='line' id='LC84'>By default Powerline caches all the statuslines and colors in a cache file in </div><div class='line' id='LC85'>the plugin&#39;s directory (or the vim directory, depending on the installation </div><div class='line' id='LC86'>method used).</div><div class='line' id='LC87'><br/></div><div class='line' id='LC88'>It&#39;s recommended that you enable the cache, as this dramatically improves vim </div><div class='line' id='LC89'>startup time after the cache file has been generated (the plugin usually loads </div><div class='line' id='LC90'>within ~100ms without the cache and ~1ms with the cache).</div><div class='line' id='LC91'><br/></div><div class='line' id='LC92'>Note: The default cache filename includes the current theme, colorscheme and </div><div class='line' id='LC93'>symbol settings in order to tie the cache file your current configuration, so </div><div class='line' id='LC94'>the cache file will be regenerated when you change any settings. This may </div><div class='line' id='LC95'>leave several old cache files in your vim folder, and these may safely be </div><div class='line' id='LC96'>deleted.</div><div class='line' id='LC97'><br/></div><div class='line' id='LC98'>Defaults: &quot;&lt;plugin_directory&gt;/Powerline_&lt;theme&gt;_&lt;colorscheme&gt;_&lt;symbols&gt;.cache&quot;</div><div class='line' id='LC99'><br/></div><div class='line' id='LC100'>------------------------------------------------------------------------------</div><div class='line' id='LC101'>4.2 Powerline_cache_enabled                          *Powerline_cache_enabled*</div><div class='line' id='LC102'><br/></div><div class='line' id='LC103'>It&#39;s possible to disable statusline caching by setting this option to 0. This </div><div class='line' id='LC104'>is mostly useful when developing statuslines.</div><div class='line' id='LC105'><br/></div><div class='line' id='LC106'>Default: 1</div><div class='line' id='LC107'><br/></div><div class='line' id='LC108'>------------------------------------------------------------------------------</div><div class='line' id='LC109'>4.3 Powerline_symbols                                      *Powerline_symbols*</div><div class='line' id='LC110'><br/></div><div class='line' id='LC111'>This option defines which set of symbols and dividers you want to use. There </div><div class='line' id='LC112'>are currently three available options: &quot;compatible&quot;, &quot;unicode&quot; and &quot;fancy&quot;.</div><div class='line' id='LC113'><br/></div><div class='line' id='LC114'>&nbsp;&nbsp;&nbsp;&nbsp;TYPE           DESCRIPTION ~</div><div class='line' id='LC115'>&nbsp;&nbsp;&nbsp;&nbsp;compatible     Doesn&#39;t use any special characters.</div><div class='line' id='LC116'>&nbsp;&nbsp;&nbsp;&nbsp;unicode        Simulates icons and arrows using similar unicode glyphs.</div><div class='line' id='LC117'>&nbsp;&nbsp;&nbsp;&nbsp;fancy          Custom icons and arrows. Requires a patched font.</div><div class='line' id='LC118'><br/></div><div class='line' id='LC119'>Symbols can be inserted into statuslines by using the following variables:</div><div class='line' id='LC120'><br/></div><div class='line' id='LC121'>&nbsp;&nbsp;&nbsp;&nbsp;VARIABLE       DESCRIPTION ~</div><div class='line' id='LC122'>&nbsp;&nbsp;&nbsp;&nbsp;$BRANCH        Inserts a branch symbol</div><div class='line' id='LC123'>&nbsp;&nbsp;&nbsp;&nbsp;$RO            Inserts a read-only symbol</div><div class='line' id='LC124'>&nbsp;&nbsp;&nbsp;&nbsp;$FT            Inserts a filetype symbol</div><div class='line' id='LC125'>&nbsp;&nbsp;&nbsp;&nbsp;$LINE          Inserts a line number symbol</div><div class='line' id='LC126'><br/></div><div class='line' id='LC127'>Default: &quot;compatible&quot;</div><div class='line' id='LC128'><br/></div><div class='line' id='LC129'>------------------------------------------------------------------------------</div><div class='line' id='LC130'>4.3.1 Compatible symbols                        *Powerline-symbols-compatible*</div><div class='line' id='LC131'><br/></div><div class='line' id='LC132'>These symbols will work in any configuration, and do not require a special </div><div class='line' id='LC133'>font to work. This option will replace the fancy icons with plain text, and </div><div class='line' id='LC134'>the pointy dividers with straight lines.</div><div class='line' id='LC135'><br/></div><div class='line' id='LC136'>------------------------------------------------------------------------------</div><div class='line' id='LC137'>4.3.2 Fancy symbols                                  *Powerline-symbols-fancy*</div><div class='line' id='LC138'><br/></div><div class='line' id='LC139'>These symbols require a custom font to work. A font patcher is provided for </div><div class='line' id='LC140'>adding the required symbols to any outline font and some bitmap fonts, see </div><div class='line' id='LC141'>|Powerline-fonts| and the provided README file for usage details.</div><div class='line' id='LC142'><br/></div><div class='line' id='LC143'>------------------------------------------------------------------------------</div><div class='line' id='LC144'>4.4 Powerline_theme                                          *Powerline_theme*</div><div class='line' id='LC145'><br/></div><div class='line' id='LC146'>TODO</div><div class='line' id='LC147'><br/></div><div class='line' id='LC148'>------------------------------------------------------------------------------</div><div class='line' id='LC149'>4.5 Powerline_colorscheme                              *Powerline_colorscheme*</div><div class='line' id='LC150'><br/></div><div class='line' id='LC151'>TODO</div><div class='line' id='LC152'><br/></div><div class='line' id='LC153'>==============================================================================</div><div class='line' id='LC154'>5. Fonts                                                     *Powerline-fonts*</div><div class='line' id='LC155'><br/></div><div class='line' id='LC156'>TODO</div><div class='line' id='LC157'><br/></div><div class='line' id='LC158'>==============================================================================</div><div class='line' id='LC159'>6. Customization                                     *Powerline-customization*</div><div class='line' id='LC160'><br/></div><div class='line' id='LC161'>------------------------------------------------------------------------------</div><div class='line' id='LC162'>6.1 Basics                                             *Powerline-cust-basics*</div><div class='line' id='LC163'><br/></div><div class='line' id='LC164'>TODO</div><div class='line' id='LC165'><br/></div><div class='line' id='LC166'>------------------------------------------------------------------------------</div><div class='line' id='LC167'>6.2 Functions                                       *Powerline-cust-functions*</div><div class='line' id='LC168'><br/></div><div class='line' id='LC169'>TODO</div><div class='line' id='LC170'><br/></div><div class='line' id='LC171'>==============================================================================</div><div class='line' id='LC172'>7.   <h2>Markdown Cheat Sheet</h2>

  <div class="cheatsheet-content">

  <div class="mod">
    <div class="col">
      <h3>Format Text</h3>
      <p>Headers</p>
      <pre>
# This is an &lt;h1&gt; tag
## This is an &lt;h2&gt; tag
###### This is an &lt;h6&gt; tag</pre>
     <p>Text styles</p>
     <pre>
*This text will be italic*
_This will also be italic_
**This text will be bold**
__This will also be bold__

*You **can** combine them*
</pre>
    </div>

      <h3>Lists</h3>
      <p>Unordered</p>
      <pre>

* Item 2
  * Item 2a
  * Item 2b</pre>
     <p>Ordered</p>
     <pre>
1. Item 1
2. Item 2
3. Item 3
   * Item 3a
   * Item 3b</pre>
